Acerca de mí

Hola, soy Gonzalo Martínez, ingeniero mecánico especializado en la gestión de datos maestros de materiales de la industria Oil & Gas.

Este proyecto fue desarrollado como parte del curso de CoderHouse que estoy finalizando, con el objetivo de implementar lo aprendido en una herramienta que combina la programación en Python con la gestión de una base de datos y la herramienta web Django.

La web constituye la base de datos de clientes del gimnasio ficticio Training Center, a la cual puede acceder el personal administrativo, con un Usuario y Contraseña personales, para registrar nuevos clientes, visualizar la ficha de un cliente y modificar o eliminar clientes.
